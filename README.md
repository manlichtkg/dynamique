# Accueil / Teste3

Ce projet est une application web construite avec [Vite](https://vitejs.dev/), [React](https://react.dev/), et [TypeScript](https://www.typescriptlang.org/), stylisée avec [Tailwind CSS](https://tailwindcss.com/).

## Prérequis

Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre machine.

## Installation

1. Clonez le dépôt ou téléchargez les fichiers sources.
2. Ouvrez un terminal dans le dossier du projet (`d:\sc04\Accueil`).
3. Installez les dépendances nécessaires :

```bash
npm install
```

## Lancer le projet en local

Pour démarrer le serveur de développement :

```bash
npm run dev
```

Une fois lancé, ouvrez votre navigateur et accédez à l'URL indiquée (généralement `http://localhost:5173`).

## Construire pour la production

Pour créer une version optimisée pour la production :

```bash
npm run build
```

Vous pouvez prévisualiser la version de production localement avec :

```bash
npm run preview
```

## Technologies utilisées

- **React** : Bibliothèque UI
- **Vite** : Outil de construction rapide
- **TypeScript** : Superset JavaScript typé
- **Tailwind CSS** : Framework CSS utility-first
- **React Router** : Gestion du routage
